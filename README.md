# Mega Mart E-Commerce Sales Dashboard

This project analyzes Mega Mart e-commerce sales data and visualizes key business insights using Power BI.

## 📊 Dashboard Insights

The dashboard provides insights into:

* Total Sales Amount: **438K**
* Total Quantity Sold: **5615**
* Total Profit: **37K**
* Average Order Value (AOV): **121K**

## 📈 Key Visualizations

* Profit by Month
* Profit by State
* Quantity by Category
* Profit by Sub-Category
* Top Buyers
* Quantity by Payment Mode

## 🛠 Tools Used

* Power BI
* Data Visualization
* Data Analysis

## 🚀 Project Goal

The goal of this project is to analyze e-commerce sales data and uncover useful insights such as profitable months, best-selling categories, and customer purchasing behavior.

This dashboard demonstrates how data visualization can help businesses understand sales trends and make better data-driven decisions.
